# How to make changes to your Website in Markdown
By Yahya Yanouri



Changing your website in Markdown may seem *quite* difficult at first. However, it is actually quite easy & can be done quickly once you understand how to utilize the language.

>**What you need to make changes in markdown is:**

>1. The website in question you are trying to make changes to

>2. A computer through which you can modify your website



>**This is the basic list of Markdown Language:**

>1. Headings: 

              # text1

              ## text2
              
              ### text3

>2. **Bold:** ** text **

>3. *Italics:* * text *

>4. Blockquote: >text

>5. Ordered List

     1. text1
     
     2. text2
     
     3. text3
     
>6. Unordered List
     
     • text1
     
     • text2
     
     • text3
     
>7. Code: ' Code '

>8. Horizontal Rule: ---

>9. Link: [  text  ](  https://www.website.com  )

>10. Image: ! [  image  ](  image.png  )
